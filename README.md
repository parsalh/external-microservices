External Microservices
===

This project is a standalone microservice designed to handle communications with external APIs for the StreetFoodGo application.

---

### **How to run:**

1. Clone the repository:
   
   ```
   git clone https://github.com/parsalh/external-microservices.git
   ```
  
2. Run the following command:
   
   ```
   ./mvnw spring-boot:run #Linux
   ```
   
3. The service will start on port `8081`!
